## Filosofia

- **Codigo abierto** — cualquier persona puede revisar, auditar y contribuir
- **Sin instaladores** — ejecutables directos o APKs que no modifican el sistema sin permiso
- **Sin root** — todas las herramientas Android funcionan sin root
- **Mejoradas con IA** — uso inteligencia artificial como asistente de desarrollo para garantizar mayor calidad y menos errores
- **Resultados reales** — cada herramienta nace de una necesidad concreta
